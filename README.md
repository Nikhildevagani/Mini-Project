# Resume–Job Match Scorer (Explainable AI)

## Project Overview

This project evaluates how well a candidate's resume matches a job description using Natural Language Processing and Machine Learning techniques.

## Problem Statement

Recruiters spend significant time manually screening resumes. This project automates resume evaluation by calculating a matching score and identifying missing skills.

## Features

* Resume and Job Description Analysis
* Match Score Calculation
* Skill Extraction
* Missing Skill Identification
* Explainable Recommendations

## Technologies Used

* Python
* NLP
* Machine Learning
* Scikit-learn
* Pandas

## Outcome

The system helps candidates improve their resumes and assists recruiters in faster candidate screening.
